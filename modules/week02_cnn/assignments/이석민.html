이석민 - Week 02 CNN 실습 결과

실습 개요

Week 02 CNN 관련 실습을 진행하며 생성된 결과물들을 정리한 문서입니다.

실습 결과

1. 기본 이미지 처리 결과

2. 컬러 스페이스 변환 데모

3. 픽셀 배열 시각화

실습 내용 설명

기본 이미지 처리

  • 이미지 로딩 및 기본 변환 작업
  • 다양한 이미지 처리 기법 적용

컬러 스페이스 변환

  • RGB, HSV, Lab 등 다양한 컬러 스페이스 간 변환
  • 각 컬러 스페이스의 특성 분석

픽셀 레벨 분석

  • 이미지를 픽셀 배열로 표현
  • 픽셀 값의 분포 및 특성 시각화

학습 성과

이번 실습을 통해 다음과 같은 내용을 학습했습니다:

  • 기본적인 이미지 처리 기법
  • 컬러 스페이스의 이해
  • 픽셀 레벨에서의 이미지 분석
